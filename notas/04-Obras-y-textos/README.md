## Criterio

- Esta carpeta reune notas centradas en una obra, dialogo, tratado o texto puntual.
- Se separa de `02-Autores/` para no mezclar fichas personales con comentarios de obras.
- Se separa de `03-Conceptos/` para no mezclar textos fuente con nociones generales.
- `Mapa.md` queda fuera por ahora, porque funciona mejor como indice o red de enlaces.

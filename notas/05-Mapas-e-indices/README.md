# Mapas e indices

## Criterio

- Esta carpeta reune notas que sirven como mapa relacional, indice o referencia transversal.
- No se mezclan con `03-Conceptos/`, porque su funcion principal no es explicar una nocion sino conectar otras notas.
- No se mezclan con `04-Obras-y-textos/`, porque no son comentarios de una obra puntual.

## Contenido esperado

- mapas conceptuales
- indices de lectura
- redes de enlaces entre autores, obras o materias

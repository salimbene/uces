# UCES

Repositorio de apuntes, materiales de cursada y documentos academicos.

## Estructura

- `notas/`: notas Markdown, autores, conceptos, obras, mapas e indices del repositorio anterior de notas.
- `2024/`, `2025/`, `2026/`: materiales organizados por anio, materia y tipo de archivo.
- `administrativo/`: documentos generales de carrera, certificados, formularios y comprobantes.

## Git y OneDrive

La carpeta esta ubicada en OneDrive para sincronizar todo el contenido. GitHub versiona apuntes, transcripciones, documentos livianos y la estructura. Bibliografia pesada, presentaciones, audios, videos y archivos grandes quedan en OneDrive pero excluidos por `.gitignore`.
